# Abstract Factory

## 実装
- 何らかの種類ごとにProductを生成したい際に用いるPattern
    - Type AのProduct1, 2, 3
    - Type BのProduct1, 2, 3
    - . . . 
- 種類ごとにProduct interfaceを実装したClassとそれを生成するFactory interfaceを実装したFactory Classを作成する
- 使用者は上記のinterfaceを介してProductとFactoryを扱う
